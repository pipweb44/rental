# العقارات الفاخرة 👑

موقع إلكتروني فاخر ومتطور لإيجار أرقى العقارات في المملكة العربية السعودية، مبني باستخدام Django مع تصميم فاخر (Luxury Style) وواجهة مستخدم راقية ومتجاوبة.

## ✨ المميزات الفاخرة

### 🎨 التصميم الفاخر (Luxury Style)
- 👑 تصميم راقي بألوان ذهبية وأنيقة
- ✨ أنيميشن متطورة وتأثيرات بصرية فاخرة
- 🌟 واجهة مستخدم راقية مع تدرجات لونية فاخرة
- 💎 عناصر تفاعلية بتصميم premium
- 🎭 وضع مظلم/مضيء فاخر مع حفظ التفضيل

### للعملاء المميزين
- 🔍 البحث المتقدم في أرقى العقارات
- 📱 واجهة مستخدم فاخرة ومتجاوبة
- 📋 طلب إيجار العقارات الفاخرة
- 📊 متابعة حالة الطلبات بأسلوب راقي
- 👤 إدارة الملف الشخصي بتصميم فاخر
- ⭐ نظام المفضلة والمشاركة

### لملاك العقارات الفاخرة
- ➕ إضافة طلبات عرض العقارات الفاخرة
- 📸 رفع صور متعددة بواجهة أنيقة
- 📈 متابعة حالة العقارات بلوحة فاخرة
- 💬 استقبال طلبات الإيجار
- 📊 إحصائيات شخصية متقدمة
- 🏆 عرض العقارات بأسلوب premium

### للإدارة الراقية
- 🎛️ لوحة إدارة فاخرة ومتقدمة
- ✅ مراجعة وقبول/رفض طلبات العقارات
- 🤝 إدارة طلبات الإيجار بواجهة راقية
- 👥 إدارة المستخدمين
- 📊 إحصائيات شاملة وتقارير فاخرة

## 🛠️ التقنيات المستخدمة

- **Backend**: Django 5.2.5
- **Frontend**: Bootstrap 5.3, HTML5, CSS3 (Luxury Style), JavaScript
- **Database**: SQLite (قابل للتغيير)
- **Icons**: Font Awesome 6.4
- **Fonts**: Google Fonts (Tajawal, Amiri - خطوط عربية فاخرة)
- **Animations**: AOS (Animate On Scroll)
- **Design**: Luxury UI/UX with Gold & Dark Theme

## 🚀 التثبيت والتشغيل

### المتطلبات
- Python 3.8+
- pip

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd rental
```

2. **إنشاء بيئة افتراضية**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# أو
venv\Scripts\activate  # Windows
```

3. **تثبيت المتطلبات**
```bash
pip install django pillow
```

4. **تطبيق قاعدة البيانات**
```bash
python manage.py migrate
```

5. **إنشاء مستخدم إداري**
```bash
python manage.py createsuperuser
```

6. **إنشاء بيانات تجريبية (اختياري)**
```bash
python create_sample_data.py
```

7. **تشغيل الخادم**
```bash
python manage.py runserver
```

8. **فتح الموقع**
```
http://localhost:8000
```
